INSTALACIÓN
===========

1. Copiar todo el contenido de este directorio en /var/www/
2. Importar la base de datos desde PHPMyAdmin, el fichero SQL se encuentra en 
   config/sql/bid_and_sell.sql.

La base de datos contiene un conjunto de usuarios y productos. Para ver qué 
usuarios están registrados, y sus contraseñas de acceso, consultar el fichero 
config/sql/Users.txt.
